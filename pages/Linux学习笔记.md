		- centos7设置中文man
			- 查找man中文安装包
				- yum list |grep man.*zh
			- 执行安装命令
				- yum install man-pages-zh-CN.noarch

				- vi .bashrc

				- 在命令行执行，为了使刚刚添加的alias生效
					- source .bashrc
			- 将/etc/locale.conf文件修改为LANG="zh_CN.UTF-8"
